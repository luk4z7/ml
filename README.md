## Install

### prerequisites
- python 2
  - numpy
  - pandas
  - sklearn
  - matplotlib
  - ipython
  - scikit-image
  - tensorflow 
  - keras
  - h5py
  - skimage
  - jupyter

clone the project
```bash
git clone https://github.com/luk4z7/ml
```

Install dependencies
```bash
pip install -r requirements.txt --user
```

Execute notebook jupyter
```bash
cd ml

jupyter notebook --allow-root
```
